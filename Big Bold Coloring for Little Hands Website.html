✅ Step 2: How to Put It Live (Free via GitHub Pages)

📌 Option 1: GitHub Pages (recommended for free, public hosting)

  1. Create a GitHub Account:
    Go to https://github.com/ and sign up.

  2. Create a New Repository:

    • Click the “+” at the top right → New Repository

    • Name it something like: coloring-book-site

    • Make it public, and check “Add a README”

  3. Upload the Code:

    • Click “Add file” → “Create new file”

    • Name the file index.html

    • Paste in the HTML code above

    • Scroll down and click “Commit changes”

  4. Enable GitHub Pages:

    • Go to Settings → Pages

    • Under "Source", choose “Deploy from a branch”

    • Set it to the main branch and root folder (/)

    • Click Save

  5. Your Site is Live!
    In a few moments, you’ll get a public link like:
    https://yourusername.github.io/coloring-book-site

✅ What You’ll Need:

  • A Netlify account (free): https://www.netlify.com

  • Your HTML file saved as index.html on your computer

🚀 Step-by-Step: Deploy Your Site on Netlify

✅ Step 1: Save Your HTML File

  1. Open Notepad (Windows) or TextEdit (Mac).

  2. Paste the website code I gave you earlier.

  3. Save it as index.html

    • On Windows: File → Save As → name it index.html and select “All Files” under file type.

    • On Mac: Make sure to use plain text, then save it as index.html.

✅ Step 2: Create a Folder for the Site

  1. Create a new folder on your desktop called something like coloring-website.

  2. Move your index.html file into that folder.

✅ Step 3: Create a Netlify Account

  1. Go to https://www.netlify.com

  2. Click Sign Up (you can use GitHub, Google, or email)

  3. After signing in, click “Add new site” → “Deploy manually”

✅ Step 4: Upload Your Site

  1. Drag and drop the folder (coloring-website) directly into the upload box.

  2. Netlify will automatically upload and deploy your site.

  3. Within seconds, you’ll see:

    • 🎉 “Site deployed successfully!”

    • A link to your live website like:
      https://whimsical-panda-12345.netlify.app

✅ Step 5: Customize Your Site Name (Optional)

  1. On your site dashboard, click “Site Settings”

  2. Under “Site name”, click “Change site name”

  3. Rename it to something like big-bold-coloring

Your link now becomes:
https://big-bold-coloring.netlify.app

💡 Extra Tips:

  • Want to make changes later? Just update your index.html and drag the folder back into Netlify again.

  • Want to add images or more pages later? Just add them to the same folder and update your HTML.
